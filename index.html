timer message

Таймер расчитан на время 23:59:59, необходимо учитывать это при вводе времени для обратного отсчета.

На всякий случай: ссылка на исходный код

В поле input вводится двузначное число. Время устанавливается кнопками "+ h"(добавить часы), "+ m"(добавить минуты), "+ s"(добавитьсекунды). После установки времени для запуска отсчета нажать "start". Для паузы нажать "pause", для продолжения отсчета нажать "resume". Для полного сброса используется "reset". Во время работы таймер выдает сообщения о своем состоянии(countdown...), выводит ошибку ввода, если в input будет строка, ошибку связанную с переполнением переменной time(когда время больше 23:59:59 - Error: time out of range). После окончания отсчета таймер сообщает, что отсчет окончен(countdown is over) и выдает короткий звуковой сигнал.
